# ObjectField

## Description

Vue binding for [createObjectField](https://core.formilyjs.org/api/models/form#createobjectfield) in `@formily/core`. It links object-shaped values to layout or input components, and follows [IFieldFactoryProps](https://core.formilyjs.org/api/models/form#ifieldfactoryprops).

The `name` prop is required. Compose child components through scoped slots.

::: demo
api/components/object-field
:::

## API

Identical to the [Field API](/en/api/components/field#field-props).
